## Использование
1. Склонировать репозиторий
```
git clone git@github.com:GKoil/frontend-project-lvl1.git
```
2. Установить зависимости
```
make install
```
3. Сгенерировать пакет локально
```
make publish
```
4. Установить пакет
```
npm link
```
